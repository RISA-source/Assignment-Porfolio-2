# Assignment-Portfolio-2

Single-page personal portfolio website with modern dark theme and video background hero section.

## Features

- Video background hero section with multiple format fallbacks
- Responsive hamburger menu for mobile devices
- Interactive skill cards with external links to documentation
- Contact form with client-side validation
- Video feedback on form submission
- Smooth scrolling navigation

## Technologies

- HTML5
- CSS3 with Grid and Flexbox layouts
- Vanilla JavaScript for form handling and mobile menu
- Google Fonts (Barrio, Barriecito, Outfit)
